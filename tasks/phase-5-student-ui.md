# Фаза 5: Студентский UI

> Статус: Ожидает | Оценка: 3–4 дня | Зависит от: Фаза 4

## Чеклист

### Layout студента
- [ ] `app/[locale]/(student)/layout.tsx` — sidebar на десктопе
- [ ] Sidebar навигация: Курсы, Прогресс, Сертификаты, Профиль, Выход
- [ ] Header: переключатель языка (ru/kz), аватар пользователя
- [ ] Mobile: bottom navigation (4 иконки)
- [ ] Dark mode поддержка (next-themes)

### Каталог курсов
- [ ] `GET /api/courses` — список с прогрессом текущего пользователя
- [ ] Страница `/[locale]/courses`
- [ ] Карточка курса: обложка, название, описание, прогресс %, уровень, кол-во видео, длительность
- [ ] Фильтр по категории (dropdown)
- [ ] Фильтр по статусу: В процессе / Завершён / Все
- [ ] Поиск по названию (debounced)
- [ ] Empty state при отсутствии курсов

### Страница курса
- [ ] `GET /api/courses/:id` — детали + список видео с прогрессом
- [ ] Страница `/[locale]/courses/:id`
- [ ] Hero: обложка, название, описание, прогресс бар
- [ ] Список видео: галочка ✓ для завершённых, current_time для незавершённых
- [ ] Кнопка "Продолжить" → последнее незавершённое видео
- [ ] Кнопка "Начать сначала" (если всё завершено)

### Страница видео (интеграция с Фазой 4)
- [ ] Полный Layout с плеером, названием, описанием
- [ ] Навигация между видео (prev/next)
- [ ] Прогресс курса обновляется после завершения видео

### Профиль студента
- [ ] Страница `/[locale]/profile`
- [ ] Форма редактирования: имя, фамилия, телефон, биография
- [ ] Загрузка аватара (Supabase Storage, бакет `avatars`)
- [ ] Email — только для чтения
- [ ] Секция смены пароля (отдельная форма)
- [ ] Статистика: общее время просмотра, завершённых курсов, прогресс

### Переключение языка
- [ ] Компонент LanguageSwitcher (ru / kz)
- [ ] Сохранение в localStorage
- [ ] `PUT /api/profile` — сохранить язык в профиле
- [ ] Автоопределение из `Accept-Language` заголовка при первом входе

### Проверка
- [ ] Все страницы переведены (ru/kz)
- [ ] Мобильная адаптация (bottom nav, full-width плеер)
- [ ] Dark mode работает
- [ ] Прогресс курсов отображается корректно
- [ ] Фильтры и поиск работают
